# Changelog

## 0.1.0-alpha.1

- Primo **installer desktop Windows** (Electron + Python embedded)
- Trascrizione offline con NVIDIA NeMo Parakeet
- Riassunto locale Qwen (RAM ≥ 16 GB) o provider cloud
- Rebrand da Sbobinator a **Qwibo**
- Documentazione su [qwibo.github.io/docs](https://qwibo.github.io/docs/)

### Limiti noti (alpha)

- Installer **non firmato** — avviso SmartScreen
- ASR solo CPU nel build desktop
- Wizard modelli (miglioramenti UI progress bar in roadmap)
- Mac non ancora disponibile
